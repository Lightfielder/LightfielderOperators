# Lightfielder Operators | Uninstalling Ops

Listed below are the steps for removing Ops from your computer.

### Remove the Ops Folder

At this point we are ready to remove the Ops folder from your hard disk to complete the Ops toolset's uninstall process.

1. Quit all Lightfielder applications

2. Navigate to your home folder in a desktop folder browsing window

3. You can (optionally) zip compress a copy of the current OPs folder to save a back up version of the files as they exist at this moment. This might be a useful step.

4. To permanently remove the `$HOME/Ops/` folder, drag the "Ops" folder directly to your trash can to delete the files.
